##  🚀 **Tecnologias**
**Este projeto foi desenvolvido com as seguintes tecnologias:**
 - HTML | CSS | JAVASCRIPT
 - ANGULAR

## :arrow_right: Como Usar

    # Clone o repositório.
    $ git clone https://github.com/thmata/EcommerceAngular
    
    # Entrar dentro da pasta clonada.
    $ cd EcommerceAngular
    
    # Instalando depedências.
    $ npm install
    
    # Rodando
    $ npm start
